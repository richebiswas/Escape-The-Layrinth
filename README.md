# 🔮 Escape The Labyrinth | Horror Escape Room Game

## 🧩 Overview

**Escape The Labyrinth** is a browser-based horror escape room game where players must solve puzzles, uncover hidden clues, and escape a mysterious laboratory before time runs out.

This project is built using **HTML, CSS, and JavaScript**, focusing on immersive UI, interactive gameplay, and suspenseful design.

---

## 🎮 Features

* 🔐 Multiple game screens (Start, Story, Game, End)
* ⏳ Countdown timer for each level
* 🧠 Puzzle-solving gameplay (passwords & patterns)
* 🔍 Interactive objects with hidden clues
* 📦 Inventory system to track found items
* 💾 Save & Load game functionality
* 🎧 Horror-themed UI with immersive fonts and effects

## 🚀 How to Run the Project

1. Download or clone the repository:

   ```bash
   git clone https://github.com/your-username/escape-the-labyrinth.git
   ```

2. Navigate to the project folder:

   ```bash
   cd escape-the-labyrinth
   ```

3. Open `index.html` in your browser.

---

## 🕹️ How to Play

* Click **START NIGHTMARE** to begin
* Read the story and enter the game
* Explore the room and:

  * Click glowing objects 🔍
  * Collect clues 📦
  * Solve puzzles 🧠
* Enter the correct password to escape
* Beat the timer ⏳ or lose!
